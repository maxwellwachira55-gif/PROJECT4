# Alex Maingi – Insurance Claims Lawyer Landing Page

This folder contains a single-page marketing site for **Advocate Alex Maingi**, an insurance claims specialist with over 10 years of experience and his own law firm.

## Structure

- `index.html` – Main page layout and content.
- `styles.css` – Visual design, layout, colours (white & blue theme), and responsive styling.
- `script.js` – Navigation behaviour, scroll-reveal animations, and WhatsApp contact helper.

## Features

- Clear sections for About, Practice Areas, Process, Testimonials, FAQ, and Contact.
- Prominent **WhatsApp call-to-action** (using number `0768 415 720`, formatted for WhatsApp links).
- Responsive layout that works on desktop, tablet, and mobile.
- Light, professional white-and-blue colour palette with subtle gradients and shadows.

## Usage

Open `index.html` directly in your browser. No additional dependencies or build steps are required.
